# Computer Vision Course

This repository contains six Computer Vision laboratory projects completed at ITMO University, primarily implemented in MATLAB.

The projects cover histogram analysis, contrast enhancement, geometric transformations, noise filtering, edge detection, image segmentation, Hough Transform, and mathematical morphology. CIE Lab, k-means, and watershed techniques are also applied to segment, detect, and separate objects.

## Laboratory Projects

| Lab | Title | Description |
| --- | --- | --- |
| 1 | Histograms, Profiles, and Projections | Image intensity analysis, contrast enhancement, barcode profiles, and object projections. |
| 2 | Geometric Image Transformations | Translation, reflection, scaling, rotation, affine and projective transformations, distortion correction, and image stitching. |
| 3 | Image Filtering and Edge Detection | Noise simulation, linear and nonlinear filtering, and edge detection with Roberts, Prewitt, Sobel, Laplacian, and Canny operators. |
| 4 | Image Segmentation | Thresholding, color segmentation, CIE Lab, nearest-neighbor classification, k-means clustering, and texture analysis. |
| 5 | Hough Transform | Classical and probabilistic line detection and circular Hough Transform for detecting circles. |
| 6 | Morphological Image Analysis | Erosion, dilation, opening, closing, object separation, and watershed segmentation. |

## Repository Contents

- `report.pdf`: final laboratory report with implementation details and experimental results.
- `lab*.mlx`: MATLAB Live Script, where the original executable file is available.
- Image files: inputs used by the corresponding MATLAB experiments.

Labs 3, 4, and 6 are preserved as complete reports because their standalone source files and some original input images are not available in the current course archive. The reports include the implementation, visual results, answers, and conclusions.
